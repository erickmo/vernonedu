# VernonEdu

Monorepo platform entrepreneurship education VernonEdu.

## Projects

| Folder | Stack | Deskripsi |
|--------|-------|-----------|
| `api/` | Go + Chi + CQRS | Backend REST API |
| `app-entrepreneur/` | Flutter Web PWA | Aplikasi siswa entrepreneurship |
| `app-dashboard/` | Flutter Web | Dashboard admin & instruktur |

## Quick Start

### API
```bash
cd api
make infra-up
make migrate-up
make dev
```

### App Entrepreneur
```bash
cd app-entrepreneur
make run-dev
```
